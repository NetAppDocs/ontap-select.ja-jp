= Deploy管理者パスワードの変更

コマンドラインインターフェイスを使用して、 Deploy 仮想マシンの管理者アカウントのパスワードを必要に応じて変更できます。

.手順
. 管理者アカウントを使用して、 Deploy ユーティリティの CLI にサインインします。
. パスワードを変更します :'password modify
. 環境に応じて、すべてのプロンプトに応答します。
