= ONTAP Select容量プールライセンスのサイズ設定に関する考慮事項
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
容量プールライセンスのライセンス容量は、最小2 TiBのTiB単位で計算されます。ライセンス容量は、使用可能なスペースの量にオーバーヘッド係数を掛けて算出します。オーバーヘッド係数は、シングルノード クラスタとマルチノード（HAペアベース）クラスタで異なります：

* シングルノード クラスタのオーバーヘッド係数は1.13です
* マルチノード クラスタのオーバーヘッド係数は2.67です


展開に十分な容量があることを確認するには、使用可能なスペースに基づいてライセンス容量を計算する必要があります。

[cols="25,30,45"]
|===
| 説明 | シングルノード | HAペアまたはHAとシングルノードの混合 


| 使用可能スペース | 20 TiB | シングルノード クラスタあたり10 TiB、HAペアの場合は20 TiB 


| ライセンス スペース | 23 TiB（20 x 1.13 = 22.6、次の完全な TiB に切り上げ） | 54 TiB（20 x 2.67 = 53.4、次の完全な TiB に切り上げ） 


| ノードあたりの最大使用可能スペース *注*：ノードまたはHAペアあたりの最大管理スペースは400 TiBです | 353 TiB（400／1.13）  a| 
シングルノード クラスタあたり：

74 TiB［（400 / 2）/ 2.67］

HAペアクラスタごと：

148 TiB（2 x 74）

|===
